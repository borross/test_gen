# test_gen
Скрипт собирает итоговый тест из одного или нескольких markdown-файлов с тестами. Поддерживает три типа вопросов, перемешивание вариантов ответов, фильтрацию по сложности и типу, весовое распределение вопросов по темам. Автоматически формирует блок правильных ответов — в том числе развёрнутых текстовых ответов на открытые вопросы.
