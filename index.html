Основы информационной безопасности

Итоговый тест. Собран за секунды.

Загрузите банки вопросов в markdown — TestGen проверит их, распределит вопросы по темам, перемешает варианты ответов и соберёт столько вариантов контрольной, сколько нужно вашим группам.

Перетащите файлы .md сюда

Один файл может содержать несколько тестов — заголовки «#» разделят их автоматически

База вопросов

Вес определяет долю вопросов из файла в итоговом тесте. Если файл содержит несколько тестов, вес делится между ними поровну.
